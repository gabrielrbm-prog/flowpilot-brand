Anatomia — Stealth

Decaimento de Opacidade

O conceito Stealth usa uma progressão de opacidade decrescente. Cada camada seguinte é 50% menos visível — criando profundidade sem ruído. O ponto cyan é a única cor viva.

  • Chevron 1: opacity 0.35 — Presente, visível
  • Chevron 2: opacity 0.20 — Sutil, secundário
  • Chevron 3: opacity 0.10 — Quase invisível, ambiental
  • Pilot Node: cyan a 50% — único ponto de cor
  • Sem glow, sem sombra — pureza absoluta
  • Hairline strokes: 0.75px a 1.5px máximo

Faça
  • Hairline strokes (max 1.5px)

  • Cyan apenas no Pilot Node
